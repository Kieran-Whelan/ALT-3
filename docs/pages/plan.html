Challenges

We faced a number of challenges while planning this project. The hardest part of the planning phase of this project was deciding on what dataset to anaylse.

Feasibility

In order to ensure that our project was feasible we had to chose 3 categories from our dataset of 21 categories. We chose to anaylse: Food and non-alcoholic beverages, Clothing and Footwear and Restaurants and Hotels. Analysing this whole dataset would not be possible in the time allocated as each of the 21 categories also contains subcategories - the raw data can be found in the data/raw folder of the project directory.
